# 《我的世界》帜和语语言包

此资源包为《我的世界》Java版游戏添加<u>臤徘斯</u>创作的人造语言<u>帜和</u>语。仍在翻译，尚未完成。

A *Minecraft: Java Edition* resource pack that adds Seniva language, a conlang created by Kinboise. Not complete, work in progress.

## 使用方法

1. 点击右上角绿色 `Code` 按钮；
1. 点击 `Download ZIP`；
1. 将下载的压缩包解压；
1. 将内层的 `Bavgizugoda-ru-Seniva-main` 文件夹放入游戏资源包文件夹；
1. 在游戏中启用此资源包；
1. 在游戏的语言菜单中选择 `Seniva u Latin` 语言。
